# Wireframe: ステータス更新（管理者）

## 想定
管理者の案件詳細（04）から更新する。UIはモーダルでもページ内でも良い。

## レイアウト（モーダル）
```
┌─────────────────────────────────────┐
│ ステータスを更新                  │
├─────────────────────────────────────┤
│ 案件ID：000123                      │
│ 現在：確認中                        │
│                                      │
│ [新しいステータス（ドロップダウン）] │
│   - 受付                             │
│   - 確認中                           │
│   - 見積済                           │
│   - 作業中                           │
│   - 納品済                           │
│   - 完了                             │
│                                      │
│ [更新する]  [キャンセル]            │
└─────────────────────────────────────┘
```

## 注記（仕様）
- 利用者は基本的にステータス更新しない（MVPで事故防止）
- 更新後は案件詳細の表示に反映する
