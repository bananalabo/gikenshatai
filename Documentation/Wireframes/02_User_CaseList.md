# Wireframe: 利用者（ディーラー/法人）案件一覧

## レイアウト（自社案件のみ）
```
┌─────────────────────────────────────┐
│ 上部ナビ：案件一覧 | 新規依頼 | ログアウト │
├─────────────────────────────────────┤
│ 検索（最低限）                        │
│  - 案件ID                              │
│  - ステータス（ドロップダウン）        │
│ [検索] [リセット]                    │
├─────────────────────────────────────┤
│ 案件テーブル（自社のみ）               │
│ ┌───────┬────────────┬────────────┐ │
│ │案件ID   │関連先/顧客名  │ステータス    │ │
│ ├───────┼────────────┼────────────┤ │
│ │ 000101 │顧客A         │受付            │ │
│ │ 000102 │顧客B         │作業中          │ │
│ └───────┴────────────┴────────────┘ │
│                                     │
│ [空状態：新規依頼を作成してください]   │
└─────────────────────────────────────┘
```

## ボタン/アクション
- `新規依頼`（案件作成画面へ）

## 注記（仕様）
- 利用者は自社案件のみ表示
