# Wireframe: メッセージスレッド（Inbox Thread）

## 目的
メッセージの文脈（会話）を確認し、返信できる状態にする。

## レイアウト
```
┌───────────────────────────────┐
│ 左ナビ（SaaS）                   │
│ - 案件一覧                         │
│ - 新規依頼（Userのみ）           │
│ - インボックス（active）        │
└───────────────────────────────┘

┌────────────────────────────────┐
│ メッセージ（スレッド）            │
├────────────────────────────────┤
│ 案件ID / threadId（サンプル）   │
│                                    │
│ 会話（相手/自分の吹き出し）      │
│                                    │
│ 返信欄（textarea） + 送信ボタン   │
└────────────────────────────────┘
```

## 注記（仕様）
- MVPではUIのみ（送信はプロトタイプ扱い）
- 実装では権限制御（自社案件だけ見える）を前提に設計
