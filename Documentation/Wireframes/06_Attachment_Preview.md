# Wireframe: 添付プレビュー（画像拡大 / PDFプレビュー）

## 想定（モーダル/別ビュー）
- 画像（途中写真）：拡大表示、前後移動
- PDF（見積/納品）：閲覧（埋め込み or 新規タブ）

## 画像プレビュー（途中写真）
```
┌─────────────────────────────────────┐
│ 途中写真：3/7                       │
│ [戻る] [前へ] [次へ]                │
├─────────────────────────────────────┤
│ [画像プレビュー（大）]              │
├─────────────────────────────────────┤
│ 情報欄：                            │
│ - 種類：途中写真                       │
│ - 撮影/アップロード日時               │
│ - ファイル名                           │
│                                     │
│ [ダウンロード]                      │
└─────────────────────────────────────┘
```

## PDFプレビュー（見積/納品）
```
┌─────────────────────────────────────┐
│ PDFプレビュー：見積PDF             │
│ [閉じる]                             │
├─────────────────────────────────────┤
│ [PDFビュー（埋め込み）]             │
│ （表示できない場合：新規タブ）       │
└─────────────────────────────────────┘
```

## 注記（仕様）
- 添付クリックからこのプレビューへ遷移
- 権限がない添付が開けない（認可でガード）
