# Wireframe: 管理者（元請け）案件一覧

## レイアウト（一覧＋検索＋行クリックで詳細へ）
```
┌─────────────────────────────────────┐
│ 上部ナビ：案件一覧 | ログアウト        │
├─────────────────────────────────────┤
│ 検索（最低限）                        │
│  - 案件ID                              │
│  - 関連先名                            │
│  - ステータス（ドロップダウン）        │
│ [検索] [リセット]                    │
├─────────────────────────────────────┤
│ 案件テーブル                            │
│ ┌───────┬────────┬────────────┐ │
│ │案件ID   │関連先   │ステータス    │ │
│ ├───────┼────────┼────────────┤ │
│ │ 000012 │Honda    │見積済        │ │
│ │ 000013 │損保A     │作業中        │ │
│ └───────┴────────┴────────────┘ │
│                                     │
│ [空状態：案件がありません]             │
├─────────────────────────────────────┤
│ フッター：データは最新表示             │
└─────────────────────────────────────┘
```

## 主要要素
- 検索/絞り込み（MVP最低限：案件ID、関連先、ステータス）
- 案件行クリックで `Admin Case Detail` へ

## 注記（仕様）
- 管理者は全案件閲覧可
