# Wireframe: 取引先（Vendors / Partners）

## 目的
取引先（ディーラー/法人）ごとに案件を分けて表示し、探す時間と誤アクセスを減らす。

## レイアウト（SaaS左ナビ＋メイン）
```
┌───────────────────────────────┐
│ 左ナビ                           │
│ - 案件一覧                         │
│ - 取引先（active）              │
│ - インボックス                   │
└───────────────────────────────┘

┌────────────────────────────────┐
│ 取引先ごとの案件（一覧）        │
│ - 取引先名                         │
│ - 案件数（目安）                    │
│ - [案件を見る]（各取引先へ遷移） │
└────────────────────────────────┘
```

## 差分（Admin / User）
- Admin：取引先一覧から選択して `案件一覧` へフィルタ遷移
- User：自社（自分の組織）だけを表示し `案件一覧` へ遷移
