# GIKENSHATAI

This repository contains the initial MVP concept and UI prototype for a B2B platform that unifies:

- case management (案件ID / 管理番号)
- file & photo sharing (見積 / 納品 / 途中写真)
- status updates
- an inbox/message thread UI

## Folder structure
- `Documentation/`: proposal + MVP PRD + workflows + wireframes
- `MVP/`: static HTML/CSS/JS prototype (no backend)
- `Meetings/`: meeting notes (source material)

## How to preview the MVP
- Open `MVP/index.html` in a browser.
